# Buddy

Norsk samtaleprogram i Python. Eget nevralnett og enkel minnelagring i JSON.

## Filer

- `buddy_v9.py` — chat og minne
- `buddy_consolidate.py` — rydder i minnefiler

## Kjøre

```powershell
python -m venv .venv
.\.venv\Scripts\Activate.ps1
pip install -r requirements.txt
python buddy_v9.py
```

# Musikkbibliotek

Teamoppgave fra GET Academy (Emne 2). Vi skulle lage et musikkbibliotek med MVC — dette er vår versjon.

Vanlig JavaScript, ingen rammeverk. State ligger i `Model/model.js`, HTML bygges i `View/`, hendelser i `Controller/`.

## Funksjoner

Søk, ønskeliste, legge til/redigere/slette album, innlogging.

## Kjøre

Åpne `index.html` i nettleser, eller:

```bash
npx serve .
```

Demo-brukere: `demo_user` / `demo_password` (og `demo_user2`, samme passord).
